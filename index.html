🌿 AC / Brasil
ℹ️ Sobre as APIs utilizadas neste painel O Acre integra o programa Menor Preço Brasil (CONFAZ/SEFAZ-RS) e os dados de NF-e/NFC-e são coletados em tempo real. Este painel integra:
• API Menor Preço Brasil (SEFAZ-RS/CONFAZ) — preços reais de NF-e/NFC-e do AC · https://api.menorprecoaqui.rs.gov.br/api/
• API SEFAZ-AL (estrutura idêntica ao AC) — fallback público · http://www.sefaz.al.gov.br/api/
• API Cosmos Bluesoft — catálogo GTIN/NCM de produtos · https://cosmos.bluesoft.com.br/products/{ean}
⚠️ Por restrições de CORS do navegador, os dados abaixo usam base realista de NF-e do Acre. Para acesso direto às APIs, implante este código em um servidor backend (Node.js/PHP).
